# NAS CodeWorks — Visual Identity Finalization

## Visual identity direction

Operational Clarity through Structured Operations.

NAS CodeWorks should look organized, trustworthy, practical, Arabic-first, locally understandable, visually calm, precise without being cold, and business-oriented rather than developer-oriented.

## Brand personality

- Professional
- Practical
- Clear
- Approachable
- Human

The identity must not feel cold, bureaucratic, or purely corporate.

## Logo direction

Wordmark + Geometric Operational Mark.

The mark should suggest:

Scattered Work
→ Structured Operations
→ Operational Clarity

Allowed logo directions:

- scattered points becoming structure
- units forming a clear path
- geometric mark suggesting organization
- operational center / structured clarity

Forbidden:

- code brackets
- radio waves
- sound waves
- microphone
- warehouse icon as main mark
- dashboard icon
- SaaS abstract mark
- N-only logo without operational meaning

## Color direction

Primary:

Deep Graphite / Ink Navy

Suggested:

- #17212B
- #1E2A36

Secondary:

Muted Teal / Operational Blue-Green

Suggested:

- #2E7D7A
- #3A8F8A

Backgrounds:

Warm Off-White / Soft Operational Gray

Suggested:

- #F6F4EF
- #F2F4F3

Text:

Near Black / Slate

Suggested:

- #1A1D21
- #27313A

Accent:

Restrained Amber, limited use only.

Suggested:

- #C98A2E
- #D49A3A

Avoid neon, startup gradients, generic SaaS blue, loud agency colors, and radio/media color language.

## Typography direction

Arabic-first modern sans-serif.

Reference:

- IBM Plex Sans Arabic
- Noto Sans Arabic as fallback/reference

Use regular, medium, semibold, and limited bold.

Avoid decorative Arabic fonts, heavy Kufi, thin low-readability fonts, overly technical fonts, and too many families.

## Iconography direction

Geometric Line / Soft Hybrid Icons.

Icons should represent:

- file organization
- reports
- workflow
- status
- follow-up
- scope
- outcome
- search
- classification

Avoid:

- code icons
- AI icons
- cloud SaaS icons
- radio/media icons
- warehouse-only icons
- startup icons

## Visual motifs

- scattered points becoming structure
- organized paths
- before / after
- status cards
- simple connection lines
- information layers
- simplified tables
- workflow maps

## Layout personality

Low to medium density, generous spacing, organized grid, RTL-first, calm cards, narrative flow.

Website should not look like SaaS, Portfolio, Agency, or Product Landing Page.

## Relationship with NAS FM

Decision:

NAS CodeWorks adapts with NAS FM.

It inherits:

- NAS name
- local trust
- seriousness
- geographic/local presence

It does not inherit:

- radio waves
- audio symbols
- broadcast language
- media colors if confusing
- NAS FM technical department impression

## Final identity principle

Every visual decision must answer:

Does this make NAS CodeWorks look like a team that turns daily operational chaos into structured operational clarity?

If yes, aligned.
If it makes the brand look like a program, platform, agency, software house, or radio extension, reject it.
