## Visual anti-patterns

Reject any design that looks like:

- SaaS landing page
- software company site
- digital agency site
- freelancer portfolio
- startup website
- warehouse-only solution
- NAS FM technical sub-page

## Hero anti-patterns

Do not use:

- dashboard hero
- laptop mockup
- product screenshot
- code visual
- terminal
- radio waves
- broadcast rings
- stock team photos
- developer imagery

## Hero requirement

Hero visual must communicate:

Scattered operational information
→ Operational Center
→ clearer work picture

## Final governance rule

Any CTA, navigation label, or visual element must help the visitor understand and describe the operational problem. If it pushes them toward requesting a software project, it is wrong.
