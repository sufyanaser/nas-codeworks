# NAS CodeWorks — Build Agent Rules

Use this file as an instruction header for Claude Code / Cursor / Codex.

## Mandatory behavior

- Read `docs/00_START_HERE.md` first.
- Read `docs/04-build/02_Website_Build_Execution_Prompt.md` before implementation.
- Do not redesign strategy.
- Do not rewrite approved copy.
- Do not add services.
- Do not publish Automation / Reporting price.
- Do not create a SaaS, portfolio, or generic software-company website.
- Do not use Dashboard Hero.
- Do not ask for budget in initial contact form.
- Do not commit unless explicitly instructed.

## First implementation target

Build a multi-page Arabic-first website inside `website`.

Pages:

- Home
- Services
- How We Work
- Case Studies / Problem-Solution Stories
- Contact / Problem Discovery

## Final report format

Report only:

1. What was built.
2. Files changed/created.
3. Commands run.
4. Build/check result.
5. Unresolved issues.
6. Next safe task.
